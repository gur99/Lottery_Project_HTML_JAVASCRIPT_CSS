Rules:

Number Validations:

All numbers in a table must be unique (no duplicates).
All input fields must be filled: empty fields are not allowed.
Numbers must be within the specified range:
1-37 for regular numbers.
1-7 for the strong number.

Table Limitations:

Users can add up to 5 tables for a single game.
Each table costs 300 Shekels from the user's balance.

Winning Conditions:

Matches are determined based on the numbers and the strong number.

Prize structure:

6 matching numbers + strong number: Maximum prize.
6 matching numbers: Secondary prize.
4 matching numbers + strong number: Lower prize.
Other combinations: No prize.

Lottery Results:

The results of the number selection are done completely randomly.
Results are displayed each time a form is submitted.
